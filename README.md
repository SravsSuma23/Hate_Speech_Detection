Scalable Big Data Forensic System for Detecting Cybercrimes on Social Media
Overview

This project presents a scalable multimodal hate speech detection system designed to analyze memes posted on social media platforms. Memes often combine images and text, making them difficult to classify using traditional single-modal models.

To address this challenge, the system integrates image features and textual features using deep learning models and processes large-scale data using Apache Spark and HDFS.

The system detects whether a meme is:

Hateful

Non-Hateful
